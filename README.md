# Travels

**Живое демо:** https://yakushev-sergey.github.io/Travels/

## 📦 Функционал

- ✅ Адаптивная верстка (мобильные, планшеты, десктоп)
- ✅ Анимации при скролле и наведении
- ✅ Слайдер на Swiper.js
- ✅ Валидация email (проверка на корректность ввода)
- ✅ Маска для телефона (автоматическое форматирование номера)
- ✅ Форма обратной связи

## 🛠️ Технологии

| Технология | Что реализовано |
|------------|------------------|
| **HTML5** | Семантическая верстка |
| **CSS3** | Flexbox, Grid, анимации, адаптив |
| **JavaScript** | Валидация формы, маска телефона, инициализация Swiper |
| **Swiper.js** | Слайдер с турами/отзывами |
